These are solutions for day 6 of 2025's Advent of Code competition. The problem description can be found at https://adventofcode.com/2025/day/6

But to summarize, the goal is to parse a custom-formatted text file and perform multiplication or addition problems on said file given 2 schemas. 

This solution is straightforward, and required splitting the answer separately by rows or columns to extract the integers to have the functions performed on.
